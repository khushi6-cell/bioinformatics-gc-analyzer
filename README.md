Project Overview
This project analyzes DNA sequences from a FASTA file and calculates:
Sequence length
GC content
GC content distribution
Tools Used
R
stringr package
How to Run
Download files.
Open gc_analysis.R in RStudio.
Ensure FASTA file is in same directory.
Click Source.
